### ☁️ What is AWS?

**Amazon Web Services (AWS)** is a cloud computing platform that provides on-demand infrastructure and services such as:

- Virtual machines
- Storage
- Databases
- Networking
- Security
- Monitoring
- Serverless computing

AWS allows developers to build scalable and highly available systems without managing physical hardware.

### 🧰 Core AWS Services

Some important AWS services covered:

- **IAM** – Identity and Access Management
- **EC2** – Virtual Servers
- **S3** – Object Storage
- **VPC** – Networking
- **RDS** – Managed Databases
- **Lambda** – Serverless Computing
- **CloudWatch** – Monitoring
