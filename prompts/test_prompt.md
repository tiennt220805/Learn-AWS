- 14 câu tình huống thực tế

## Yêu cầu chi tiết:

### 1. Câu lý thuyết (6 câu)

Dùng để kiểm tra kiến thức nền tảng AWS, ví dụ:

- Khái niệm các dịch vụ AWS
- AWS Global Infrastructure
- Shared Responsibility Model
- Pricing Model
- Security best practices
- So sánh tính năng dịch vụ
- Kiến thức cơ bản Cloud

Các câu này hỏi trực tiếp kiến thức.

### 2. Câu tình huống thực tế (14 câu)

Phải là dạng scenario giống đề thi AWS thật.

Tức là đưa ra tình huống doanh nghiệp / kỹ thuật và hỏi chọn dịch vụ hoặc solution như thế nào là phù hợp nhất.

Ví dụ:

- Công ty muốn giảm chi phí cho workload không ổn định
- Startup cần web app có High Availability
- Hệ thống cần lưu file an toàn và lifecycle policy
- Database cần tự động failover
- Nhân viên cần quyền truy cập tối thiểu bằng IAM
- Ứng dụng bị traffic tăng đột biến
- Ứng dụng toàn cầu cần độ trễ thấp
- Backup dữ liệu với chi phí thấp
- Chạy serverless cho tác vụ ngắn hạn

Các câu hỏi phải yêu cầu chọn giải pháp AWS tốt nhất.
